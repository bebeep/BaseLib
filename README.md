# BaseLib
Android BaseLib

v1.0 包含功能：

1、网络框架okhttp封装；

2、图片加载框架picasso;

3、上拉加载下拉刷新框架；

4、带title的网络轮播图；

5、加载动画；

6、圆形进度条，用于上传、下载文件；

7、Gson解析工具；

8、图片压缩工具；

9、一系列自定义控件

10、常用java工具类及函数。

提交到github命令：
git remote add origin git@github.com:bebeep/BaseList.git   //连接到github某个分支

git add .    //添加所有修改到分支

git commit -m "注释" //提交所有修改

git push origin master  //提交到github


//只提交修改的部分

git add -u

git commit -m "注释"

git push


//只提交某个文件的修改
